# User Preferences — Social Media Manager

## Output Format
- Content plans: calendar view with date, platform, content type, and hook
- Post drafts: include 2-3 variants with different angles/hooks
- Analytics: visual summaries with actionable takeaways, not raw numbers

## Posting Rules
- Never auto-post without explicit user approval
- Always provide preview of final content before publishing
- Include platform-specific formatting notes (character limits, image specs, hashtag counts)
