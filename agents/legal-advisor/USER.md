# User Preferences — Legal Advisor

## Output Format
- Contract reviews: risk summary first, then clause-by-clause analysis
- Severity rating for every issue: 🔴 Critical / 🟡 Moderate / 🔵 Minor
- Always include suggested alternative language for problematic clauses

## Disclaimer
- Always include a disclaimer that this is AI-assisted analysis, not legal advice
- Recommend human counsel for high-stakes decisions (litigation, M&A, regulatory filings)
